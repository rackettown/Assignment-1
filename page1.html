  • HTML (HyperText Markup Language)
    1. Standard markup language for documents and is used to be displayed in a web browser
    2. It tells the browser how to display content
  • CSS (Cascading Style Sheets)
    1. Describes how HTML elements are to be displayed
    2. Can control the layout of multiple web pages all at once
  • Semantic tags
    1. Tags used in HTML to help organize a webpage
    2. A semantic element clearly describes its meaning to both the browser and the developer
